Basics of the Web and Html

The Worldwide Web

Collection of Html Documents and links to other files (videos, text, images, and music)

  • Html = HyperText Markup Language
  • Html is what glues the web together
  • Connected via Hyperlinks which is what makes the web, weblike

Components of the Web

4 Major Components to the Web

  1. Browser - Makes the web viewable to You/Me or a Person
  2. HTTP - the main protocol of the Web
  3. The Internet (Cloud) - the world's largest computer network
  4. Servers - computers that host files that make up the web

Intro to HTML

Components of Html

  • text content (what you see)
  • markup (what it looks like)
  • references to other documents e.g. videos, images
  • links to other pages

Intro to Html Markup

Components of Html Markup

  • Opening tag - inside < >
  • Contents - in between tags
  • Closing Tag - inside
  • Element - All three together

Examples of Tags

  • b - bold
  • em - emphasis (italics)
  • a - anchor (for making links)
  • img - image, attribute is src - source (url) and alt - text displays when the server fails to load the page
  • br - line break ( void tag - no closing tag)
  • p - paragraph

Making links

Html Attributes - property of and element that modifys it e.g. href

Examples of Html Attributes

  • href
  • src
  • alt

2 Types of Elements

  • inline - functions in the line - e.g. span, strong, img, a, br
  • block - actually creates a box that can be modified height and width - e.g. div, p, form
